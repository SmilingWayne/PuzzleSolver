    mid_loop

    Aliases: Middorupu

    ðŸ“– Rule Reference: Read full rules on external site

    Input Format

    Grid of integers representing circle positions.

    • - or 0: No clue.
    • 1: Circle at cell center.
    • 2: Circle on the bottom edge.
    • 3: Circle on the right edge.
    • 5: Circle on both bottom and right edges (2+3).

    Output Format

    No output description provided.

    Examples

    Python Quick Start

    Use the following code to solve this puzzle directly:

    import puzzlekit

# Raw input data
problem_str = """
4 4
- - 1 -
- 1 - 2
3 - - -
- - - -
"""

# Solve
res = puzzlekit.solve(problem_str, puzzle_type="mid_loop")

# Print solution grid
print(res.solution_data.get('solution_grid', []))

# Visualize (optional)
res.show()
